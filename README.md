# MeetMiddle (MVP)

여러 출발지를 입력하면 **중간 지점(좌표 평균)**을 계산하고, **중간 지점 기준 1km 내 카페 5개**를 추천하는 웹앱입니다.  
지도는 Kakao Map SDK로 표시합니다.

## 기능

- 출발지 주소 입력: 2개 ~ 5개 (입력창 추가/삭제)
- 중간 지점 계산: 입력 좌표의 평균
- 지도 표시: 출발지 마커 + 중간 지점 마커
- 카페 추천: 중간 지점 기준 반경 1km, 최대 5개
- 예상 이동거리: 각 출발지 → 중간 지점 거리(m/km)

## 사용 기술

- React + Vite
- Tailwind CSS
- Kakao Map API (Geocoder/Places)
- Pretendard 폰트(웹폰트 CDN)

## 환경 변수 설정

프로젝트 루트에 `.env` 파일을 추가하고, 아래 값을 넣어주세요.

```env
VITE_KAKAO_MAP_KEY=여기에_카카오_자바스크립트_키
```

예시는 ` .env.example `를 참고하세요.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 표시되는 URL로 접속하면 됩니다.

## 사용 방법 (MVP)

1. 출발지 주소를 2개 이상 입력합니다.
2. `중간 지점 계산하기` 버튼을 누릅니다.
3. 지도에 마커가 표시되고, 아래에 중간 좌표 / 카페 추천 / 예상 이동거리가 나옵니다.

## 주의사항

- 주소 → 좌표 변환(Geocoding) 및 카페 검색(Places)은 Kakao 키가 필요합니다.
- 입력 주소가 올바르지 않거나 좌표로 변환되지 않으면 에러가 표시됩니다.
