# mannayeok.kr

만나역은 여러 출발지를 입력하면 거리 균형, 주변 상권, 노선 접근성을 함께 고려해 **실제로 만나기 좋은 역과 약속 장소**를 추천하는 웹앱입니다.  
서비스 도메인은 `mannayeok.kr` 기준으로 사용합니다.

## 기능

- 출발지 주소 입력: 2개 ~ 5개 (입력창 추가/삭제)
- 만나기 좋은 역 추천: 거리 균형, 상권, 노선 접근성을 종합해 후보 역 추천
- 지도 표시: 출발지 마커 + 추천역 마커
- 약속 장소 추천: 추천역 주변 카페, 밥집, 술집, 놀거리 확인
- 이동 정보: 각 출발지 → 추천역 거리와 지도앱 길찾기 연결

## 사용 기술

- React + Vite
- Tailwind CSS
- Kakao Map API (Geocoder/Places)
- Pretendard 폰트(웹폰트 CDN)

## 환경 변수 설정

프로젝트 루트에 `.env` 파일을 추가하고, 아래 값을 넣어주세요.

```env
VITE_KAKAO_MAP_KEY=여기에_카카오_자바스크립트_키
```

예시는 ` .env.example `를 참고하세요.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 표시되는 URL로 접속하면 됩니다.

## 사용 방법 (MVP)

1. 출발지 주소를 2개 이상 입력합니다.
2. `만나기 좋은 역 찾기` 버튼을 누릅니다.
3. 추천역, 지도, 길찾기, 주변 약속 장소 후보를 확인합니다.

## 주의사항

- 주소 → 좌표 변환(Geocoding) 및 카페 검색(Places)은 Kakao 키가 필요합니다.
- 입력 주소가 올바르지 않거나 좌표로 변환되지 않으면 에러가 표시됩니다.
